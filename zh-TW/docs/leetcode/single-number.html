136. Single Number

方法​

為了有效解決這個問題，我們可以利用 XOR 運算。 數組中所有元素的 XOR 運算會取消掉重複的數字，留下只出現一次的元素。

讓我們通過一個例子來理解這個方法：

arr = [4, 3, 2, 4, 2]

result = 0

for num in arr:
    result ^= num

print(result)

在這個例子中，數組中所有元素的 XOR 運算結果是 4 ^ 3 ^ 2 ^ 4 ^ 2，等於 3。 結果將是我們要找的那個單一數字。

實作​

位操作​

現在，讓我們將我們的方法轉化成一個 Python 函數：

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        result = 0

        for num in nums:
            result ^= num

        return result

或者，像個專業人士一樣使用 reduce lambda 函數：

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        return reduce(lambda p, n: p ^ n, nums)

結論​

在這篇文章中，我們深入探討了「單一數字」問題，討論了一種基於 XOR 的有效方法。 這個問題不僅考驗你對位操作的理解，還挑戰你跳出框架思考。
