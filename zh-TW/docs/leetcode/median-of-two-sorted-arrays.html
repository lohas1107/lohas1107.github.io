4. Median of Two Sorted Arrays

問題描述​

「兩個排序陣列的中位數」這個問題，要求是找出兩個排序陣列的中位數。 給定兩個排序的陣列，nums1 跟 nums2，大小分別為 m 跟 n， 任務就是找出這兩個排序陣列的中位數。

範例​

nums1 = [1, 3]
nums2 = [2]
輸出: 2.0

nums1 = [1, 2]
nums2 = [3, 4]
輸出: 2.5

方法​

要解這題，我們可以用二分搜尋的概念來處理。 想法是要把陣列分成兩半，讓左邊的元素都比右邊的小。 確保這個性質後，我們就能有效地找到中位數。

class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        # 確保 nums1 是較小的那個陣列
        if len(nums1) > len(nums2):
            nums1, nums2 = nums2, nums1

        m, n = len(nums1), len(nums2)
        low, high = 0, m

        while low <= high:
            partitionX = (low + high) // 2
            partitionY = (m + n + 1) // 2 - partitionX

            maxX = float('-inf') if partitionX == 0 else nums1[partitionX - 1]
            minX = float('inf') if partitionX == m else nums1[partitionX]

            maxY = float('-inf') if partitionY == 0 else nums2[partitionY - 1]
            minY = float('inf') if partitionY == n else nums2[partitionY]

            if maxX <= minY and maxY <= minX:
                if (m + n) % 2 == 0:
                    return (max(maxX, maxY) + min(minX, minY)) / 2.0
                else:
                    return max(maxX, maxY)
            elif maxX > minY:
                high = partitionX - 1
            else:
                low = partitionX + 1

解釋​

  1. 函式 findMedianSortedArrays 接受兩個排序陣列作為輸入。
  2. 它確保 nums1 是為了優化目的而較小的那個陣列。
  3. 對較小的陣列（nums1）進行二分搜尋，把它分成兩半。
  4. 計算較大陣列（nums2）對應的分割點。
  5. 確定分割點邊界上的元素。
  6. 檢查分割點是否處於找中位數的正確位置。
  7. 根據邊界元素的比較調整搜尋範圍。
  8. 最終根據條件返回中位數。

複雜度分析​

這個方法的時間複雜度是 O(log(min(m, n)))，其中 m 和 n 是兩個陣列的大小。 二分搜尋確保了有效的搜尋空間縮減。 空間複雜度為 O(1)，因為我們只使用了固定數量的額外空間。

這個解決方案提供了一個找出兩個排序陣列中位數的最優方法，滿足了時間和空間的限制。
