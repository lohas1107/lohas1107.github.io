在 Spring Boot 使用 AOP 印日誌

使用 AOP (Aspect Oriented Programming) 的方式印出日誌，會比在各處程式中寫印日誌來的簡潔， 集中管理印日誌的程式，避免影響閱讀業務邏輯。

建置 Log4j2

參考這篇建置 Log4j2 文章。

加入 Dependency

在 build.gradle 加入 Spring Boot AOP dependencies：spring-boot-starter-aop，

dependencies {
    implementation 'org.springframework.boot:spring-boot-starter-web'
    implementation 'org.springframework.boot:spring-boot-starter-aop'
    compileOnly 'org.projectlombok:lombok:1.18.18'
    annotationProcessor 'org.projectlombok:lombok:1.18.18'

    testImplementation 'org.springframework.boot:spring-boot-starter-test'
}

設定 cURL Connect Timeout 時間

發布： 2021-06-14   |   分類： 技術   |   閱讀： 72 字 ~1 分鐘

因為 curl 的預設 Timeout 時間為 5 分鐘1，所以在進行 Timeout 的測試時，要等待很長的時間，
可以透過設定 --connect-timeout 參數，以減少等待時間。

# 設定 Timeout 為 5 秒
curl http://localhost:8080 --connect-timeout 5

建立容器化 Spring Boot 專案

發布： 2021-04-30   |   分類： 技術   |   閱讀： 222 字 ~1 分鐘

建立新 Spring Boot 專案

  1. 使用 Intellij IDEA 建立新專案，專案資訊如下：

    • Spring Boot 版本：2.4.5
    • 專案類型：Gradle
    • 語言：Java 11
    ① 使用 Intellij IDEA 內的 Spring Initailizr 創建新專案② 加入 Spring Web Dependency

  2. 增加 HelloController.java 如下：

    package cc.secondbrain.demo.controller;
   
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;
   
@RestController
public class HelloController {
   
    @RequestMapping("/")
    public String hello() {
        return "Hello Spring Boot";
    }
   
}

  3. 啟動 Spring Boot 服務，用瀏覽器瀏覽 http://localhost:8080/，就可以看到「Hello Spring Boot」
