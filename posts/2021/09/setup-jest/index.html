在 WebStorm 建置 Jest 單元測試

發布： 2021-09-06   |   分類： 技術   |   閱讀： 72 字 ~1 分鐘

加入 Jest Library

WebStorm 預設沒有載入 Jest，所以會無法識別 Jest 語法，
要從 Preferences → Languages & Frameworks → JavaScript → Libraries 加入 Jest Library。

① 點擊 Download② 加入 Jest Library

用 npm 安裝 Jest

npm init   
npm install jest

設定 Jest Configuration

#npm #jest #javascript
